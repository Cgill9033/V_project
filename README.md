# Valentine's Proposal Website

A cute and simple website for a Valentine's Day proposal.

## Features
- Responsive design that works on phones and desktops
- "Yes" button shows a loving message
- "No" button playfully moves away when hovered or touched

## Usage
1. Open `index.html` in a web browser, or run a local server.
2. Click the buttons to interact.

## Customization
- To add a custom background image to the thanks page, uncomment and edit the background-image line in thanks.html.
- To add a background video, uncomment the video element in thanks.html and replace 'your-video.mp4' with your video file (ensure it's muted for autoplay).

## Troubleshooting
- If the "No" button doesn't move, ensure JavaScript is enabled.
- For mobile, touch the "No" button to make it move.
